介绍

本文档列出了 Go 生态系统中常用的编辑器插件和 IDE，它们使 Go 的开发更加高效和无缝。 这个维基提供了支持 Go 开发的编辑器和 IDE 的完整列表 。

选项

Go 生态系统提供了各种编辑器插件和 IDE，以增强您的日常编辑、导航、测试和调试体验。

  • vim: vim-go 插件为 Go 语言提供支持
  • Visual Studio Code: Go 扩展为 Go 语言提供支持
  • GoLand: GoLand 作为独立的 IDE 或作为 IntelliJ IDEA Ultimate 的插件
  • Atom: Go-Plus 是一个 Atom 包，为 Go 提供增强的支持

注意，上述只是一些最佳解决方案; Wiki 提供了更全面的社区维护的 IDE 和文本编辑器插件清单。
